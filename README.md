# 🧠 AI Health Fraud Detection System

An AI-powered application that detects potential healthcare fraud using LLMs and API-based architecture.

## 🚀 Features
- AI-based claim analysis
- Fraud risk scoring
- Explainable outputs
- Interactive Streamlit UI
- MCP-style tool integration

## 🏗 Architecture
- Frontend: Streamlit
- Backend: FastAPI
- AI Layer: OpenAI GPT
- Tool Layer: Fraud detection APIs

## 🛠 Tech Stack
Python, FastAPI, Streamlit, OpenAI, REST APIs

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Start backend:
   uvicorn server:app --reload

3. Start frontend:
   streamlit run app.py
